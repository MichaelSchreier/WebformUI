Instructions

Select any element from the Input, Radios / Checkboxes, Select and Buttons tab and drag to the form area on the left.

Note that the order in which the elemnts are placed in the form is also the order in which the inputs are passed to the script eventually.

Clicking on an element in the form area allows to edit its properties, such as name, placeholder, label and so on. Note that for SAS script the ID / Name option is passed to the script along with the value entered by the user.

The Radios / Checkboxes and Select elements have the additional field Respective arguments which define the actual command that is sent to the script. The first argument corresponds to the first option and so on.

When the form is completed go to the Save Form tab and select the script type, the name of the script to be launched and finally click Save.
